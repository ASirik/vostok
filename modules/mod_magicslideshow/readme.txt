INSTALLATION:

1. Go to the admin area of your Joomla webite.

2. Select the menu 'Extensions' > 'Install/Uninstall'.

3. In the 'Upload Package File' section click 'Browse...' and select the zip file with Joomla extension for Magic Slideshow. Then click 'Upload File & Install' button.

4. To configure Magic Slideshow, select the menu 'Extensions' - > 'Module Manager'. Click the link 'Magic Slideshow module for Joomla'.

5. Magic Slideshow is ready to use!

6. To upgrade your version of Magic Slideshow (which removes the "Please upgrade" text), buy Magic Slideshow and overwrite the ${mod.installer.upgrade.path}/magicslideshow.js file file with the new one in your licensed version.
